# Development Setup

This guide will help you set up your local development environment to contribute to Mako Code.

## Prerequisites

- Node.js (v14 or later)
- npm or yarn
- Git

## Clone the Repository

```bash
git clone https://github.com/mako-code/mako-code.git
cd mako-code
```

## Install Dependencies

```bash
npm install
# or
yarn install
```

## Development Server

To start the development server:

```bash
npm run dev
# or
yarn dev
```

## Building for Production

```bash
npm run build
# or
yarn build
```

## Running Tests

```bash
npm test
# or
yarn test
```

## Code Style and Linting

We use ESLint and Prettier to maintain code quality. You can run the linter with:

```bash
npm run lint
# or
yarn lint
```

## Submitting Changes

Please refer to our [Contribution Guidelines](./guidelines) for information on how to submit your changes.
