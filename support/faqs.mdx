# Frequently Asked Questions

## General Questions

### What is Mako Code?

Mako Code is a comprehensive development platform that helps teams build, deploy, and manage applications more efficiently. It provides tools for project management, resource orchestration, deployment, monitoring, and more.

### Is Mako Code open source?

Yes, Mako Code is open source and available under the MIT License. You can find the source code on [GitHub](https://github.com/mako-code/mako-code).

### What programming languages does Mako Code support?

Mako Code supports applications written in any programming language. The core SDK is available for JavaScript/TypeScript, Python, and Go, with more language support coming soon.

## Installation and Setup

### What are the system requirements for Mako Code?

Mako Code requires:
- Node.js v14 or later
- npm or yarn
- Git

### How do I update Mako Code to the latest version?

To update Mako Code, run:

```bash
npm update -g mako-code
# or
yarn global upgrade mako-code
```

### Can I use Mako Code in a CI/CD pipeline?

Yes, Mako Code is designed to work well in CI/CD environments. We provide integrations for popular CI/CD platforms like GitHub Actions, GitLab CI, and Jenkins.

## Features and Usage

### Does Mako Code support multi-cloud deployments?

Yes, Mako Code supports deploying to multiple cloud providers including AWS, Google Cloud, and Azure. You can even deploy the same application to different cloud providers for redundancy.

### How does Mako Code handle secrets and sensitive information?

Mako Code includes a secure secrets management system that encrypts sensitive information at rest and in transit. Secrets are never stored in plain text and are only decrypted when needed.

### Can I use Mako Code with my existing applications?

Yes, Mako Code is designed to work with both new and existing applications. You can gradually adopt Mako Code features for your existing projects.

### Does Mako Code support database migrations?

Yes, Mako Code includes tools for managing database schemas and migrations across different environments.

## Pricing and Support

### Is there a free tier for Mako Code?

Yes, Mako Code offers a free tier that includes basic features suitable for small projects and individual developers.

### What kind of support is available for Mako Code?

We offer multiple support options:
- Community support through our [Discord server](https://discord.gg/mako-code)
- Documentation and tutorials
- Email support for paid plans
- Priority support with SLAs for enterprise plans

### Can I get training for my team on how to use Mako Code?

Yes, we offer training sessions for teams. Please contact our sales team at sales@mako-code.org for more information.

## Troubleshooting

### I'm getting an "API key invalid" error. What should I do?

First, check that you're using the correct API key. If the issue persists, try regenerating your API key in the dashboard. If you're still having problems, please contact support.

### How do I report a bug in Mako Code?

You can report bugs by creating an issue in our [GitHub repository](https://github.com/mako-code/mako-code/issues). Please include as much detail as possible to help us reproduce and fix the issue.

### Where can I find logs for debugging?

Mako Code logs are stored in:
- Linux/macOS: `~/.mako/logs`
- Windows: `%USERPROFILE%\.mako\logs`

You can also view logs using the CLI:

```bash
mako-code logs
```
